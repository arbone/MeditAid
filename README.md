# MeditAid 🧘‍♂️

MeditAid è un'applicazione web progettata per aiutare gli utenti a praticare la meditazione attraverso tracce audio guidate e un timer intuitivo. L'app offre un'esperienza minimalista e user-friendly per facilitare la pratica della mindfulness.

## 🌟 Caratteristiche

- **Timer di Meditazione**: Timer personalizzabile con controlli intuitivi
- **Tracce Audio**: Tre diverse tracce di meditazione guidata:
  - Gratitudine
  - Respiro
  - Rilassamento Profondo
- **Design Responsive**: Interfaccia ottimizzata per dispositivi mobile e desktop
- **Modalità PWA**: Installabile come app nativa su dispositivi mobili
- **Audio Web API**: Gestione professionale dell'audio per una riproduzione fluida

## 🚀 Demo

Prova l'applicazione live: [MeditAid App](https://meditaid.netlify.app)

## 🛠️ Tecnologie Utilizzate

- React.js
- Vite
- CSS3

## 💻 Installazione e Utilizzo

1. Clona il repository:
```bash
git clone https://github.com/tuousername/meditaid.git
```

2. Installa le dipendenze:
```bash
cd meditaid
npm install
```

3. Avvia il server di sviluppo:
```bash
npm run dev
```

4. Build per la produzione:
```bash
npm run build
```

## 📱 Utilizzo come PWA

1. Visita l'app dal tuo dispositivo mobile
2. Seleziona "Aggiungi alla schermata Home"
3. L'app sarà installata come applicazione nativa

## 🎯 Obiettivi del Progetto

MeditAid nasce con l'obiettivo di:
- Rendere la meditazione accessibile a tutti
- Offrire un'esperienza utente fluida e intuitiva
- Fornire uno strumento gratuito per il benessere mentale
